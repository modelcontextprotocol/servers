# Servidores Model Context Protocol (MCP)

Uma coleção de Servidores MCP oficiais e suportados pela comunidade que ajudam a conectar modelos de Inteligência Artificial a serviços externos.

## O que é o MCP?
O Model Context Protocol (MCP) é um padrão aberto que permite que as IAs acessem de forma segura ferramentas e fontes de dados externas. 

## Como utilizar
Cada diretório neste repositório contém um servidor MCP distinto. Para instalar e executar um servidor específico, navegue até a pasta dele e siga as instruções no respectivo `README`.
